Student Score Analysis Dashboard

Introduction

This project analyzes student scores considering various demographic and socio-economic factors. The goal is to understand how these factors influence student performance.

The dataset includes information on students' gender, ethnic group, parents' education level, lunch type, test preparation, and scores in mathematics, reading, and writing. By analyzing this data, we aim to uncover patterns and insights that can help educators and policymakers improve student outcomes.

Project Overview

This dashboard provides an interactive and comprehensive analysis of student performance. Key areas of focus include:

  • Gender distribution and its impact on scores
  • Score distribution in mathematics, reading, and writing
  • Influence of parents' education level on student performance
  • Impact of lunch type on scores
  • Distribution of weekly study hours
